# Snake Game
## Description
Recreation of the classic Snake game using C++ and Unity. This project provides a nostalgic gaming experience with modern graphics and smooth gameplay. The game includes features like score tracking, level design, and user input handling.

## Features
* Classic Snake game mechanics with a modern twist
* Score tracking and high score display
* Multiple levels with increasing difficulty
* Smooth gameplay with user input handling
* Unity-based graphics for a visually appealing experience

## Tech Stack
* Programming Language: C++
* Game Engine: Unity
* Operating System: Windows, macOS, Linux
* Dependencies: Unity Engine, C++ Standard Library

## Installation Instructions
To run the project, follow these steps:
1. Install Unity Hub and Unity Editor (version 2022.1 or later)
2. Clone the repository using `git clone https://github.com/your-username/Snake-Game.git`
3. Open the project in Unity Editor by selecting `File` > `Open Project` and navigating to the cloned repository
4. Build the project for your desired platform (Windows, macOS, Linux) using `File` > `Build Settings`

## Usage Examples
To run the game, follow these steps:
1. Build the project for your desired platform
2. Run the executable file (e.g., `SnakeGame.exe` on Windows)
3. Use the arrow keys to control the snake and eat food pellets
4. Avoid colliding with the walls or the snake's body

## Project Structure
The project is organized into the following folders:
* `Assets`: Unity assets, including graphics, sound effects, and level designs
* `Scripts`: C++ scripts for game logic, user input handling, and score tracking
* `Scenes`: Unity scenes for each level
* `Plugins`: Unity plugins for dependencies and third-party libraries

## Configuration
To configure the game, modify the `GameSettings.cs` script in the `Scripts` folder. You can adjust settings like:
* `gridSize`: The size of the game grid
* `snakeSpeed`: The speed of the snake
* `foodSpawnRate`: The rate at which food pellets spawn
